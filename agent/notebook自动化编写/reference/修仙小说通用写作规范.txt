【修仙小说通用写作规范·去AI化要点】

一、修仙小说的"人味"从哪来
1. 主角不是全知全能：他会判断失误、会后悔、会被骗
2. 配角有自己的生活：他们不是等主角来推动剧情的NPC
3. 修仙界有烟火气：坊市里讨价还价、宗门里勾心斗角、师兄弟间的小摩擦
4. 修炼是苦差事：不是闭关就突破，更多时候是枯燥的日复一日
5. 资源有限是核心矛盾：为什么抢？因为不够分

二、去AI化的具体操作
1. 禁用词表（出现即AI味）：
   - "目光如炬""心中一震""不禁""缓缓""微微""淡淡"
   - "仿佛""宛如""犹如""恍若""霎时""须臾""刹那"
   - "蓦然""陡然""赫然""竟然""居然"
   - "宛如天神下凡""如同天降甘霖"等比喻

2. 替换方案：
   - "心中一震"→"他手里的碗差点没端住" / "脑子里嗡了一下"
   - "不禁笑了"→"嘴角翘了一下，又赶紧绷住"
   - "缓缓开口"→"他顿了顿，才说" / "他清了清嗓子"
   - "微微点头"→"他嗯了一声" / "他点了一下头"
   - "淡淡说道"→"他说" / "他随口道"

3. 句式变化：
   - 长短句交错：短句2-5字，长句20-40字
   - 不要每句都是"主语+动词+宾语"
   - 允许省略主语：中文天然支持
   - 偶尔用倒装、插说

4. 对话真实感：
   - 人说话不会完整：省略、打断、含糊
   - 口头禅和语气词：嗯、啊、吧、呢、嘛
   - 不同身份的人说话方式不同：长老端着说，师兄弟随意说
   - 修仙者也会说俗话：不是每句都文绉绉

三、修仙小说节奏参考
1. 凡人修仙传节奏：慢热型，前期铺垫多，中后期加速
2. 每章结构：一个场景为主→推进一点剧情→留个钩子
3. 大节奏：日常→冲突→战斗→收获→日常（循环）
4. 小节奏：对话推进→动作描写→心理活动→环境渲染（交替）
5. 升级节奏：不是每章都升级，积累3-5章再突破

四、修仙小说常见避坑
1. 不要让主角太顺利：韩立的成功来自谨慎和算计
2. 不要配角工具化：每个配角都有自己的故事
3. 不要设定堆砌：修炼体系要自洽，不要为了爽加设定
4. 不要战斗同质化：每场战斗要有不同的策略和手段
5. 不要忽略日常：日常是让读者喘息和建立情感连接的关键
